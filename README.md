# RentRadar Latvia

Telegram bot that monitors ss.lv 24/7 and sends you an instant alert the moment a new rental listing matches your filters.

**Bot:** [@ApartmentsLVBot](https://t.me/ApartmentsLVBot) · **Website:** [rentradar.one](https://rentradar.one)

---

## How it works

1. **Set your filters** — city, district, price range, number of rooms, floor, building series
2. **We scan ss.lv** — automatically, every 5–30 minutes, around the clock
3. **You get notified** — Telegram message with photo, price, address, and floor the moment a match appears

No account needed. Works entirely inside Telegram. Covers all of Latvia.

---

## Features

- Alerts for all Latvian cities (Riga, Jūrmala, Liepāja, Daugavpils and more)
- Multiple filters — monitor several searches at once
- Save interesting listings and add personal notes
- Pause and resume alerts anytime
- Hot listing detection — get notified when a listing is priced below market
- Price change history on any listing
- Weekly market analytics digest

**Languages:** English · Русский · Latviešu
